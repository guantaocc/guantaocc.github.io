汉诺塔问题

7/2/2021 递归

描述

经典递归问题

思路

  • 递归结束条件 只剩一个盘子从 a -> c
  • 递归过程 将 a 中的 n-1 个移动到 b 将 a 剩余的 一个 移动到 c 将 b 中的 n-1 个移动到 c

题解

function hAnotA(A, B, C) {
  // n个盘子
  let n = A.length;
  // 递归函数
  function move(n, A, B, C) {
    // 递归终止条件：只剩一个盘子从 A -> C
    if (n === 1) {
      C.push(A.pop());
      return;
    }
    // 将A 中的 n-1个移动到B
    move(n - 1, A, C, B);
    // 将 A 剩余的一个移动到C
    C.push(A.pop());
    // 将 B中的 n-1个移动到C
    move(n - 1, B, A, C);
  }
  move(n, A, B, C);
}
