## input 标签 type 类型

- button: 按钮
- checkbox
- color: 元素选择
- date
- email: 带有邮箱验证
- file: 文件上传 multiple
- hidden: 隐形控件，但受 form 控制
- password
- radio
- submit: 表单自交
- text

## textarea 如何禁止拉伸

```css
textarea {
  resize: none;
}
```
